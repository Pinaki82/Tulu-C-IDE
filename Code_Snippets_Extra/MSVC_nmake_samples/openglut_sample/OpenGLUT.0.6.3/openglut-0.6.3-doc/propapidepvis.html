OpenGLUT 1.0 glutVisibilityFunc Proposal

Note:
This is a proposal for the purpose of discussion and discussion. This API is not currently implemented in OpenGLUT.

Description

In 1996 Mark Kilgard documented the GLUT 3.6 API in the book OpenGL-on-X. glutVisibilityFunc() was described as obsolete, having been replaced by the glutWindowStatusFunc() callback.

After 8 or more years of obsolesance, and given OpenGLUT's plan to sweep out old GLUT cruft, it is time to remove this function.

glutWindowStatusFunc() provides a superset of the functionality of glutVisibilityFunc().
